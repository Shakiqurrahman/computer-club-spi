# Computer Club - SPI

## Tech Stack

**Client:** React, Tailwind CSS

## Run Locally

Clone the project

```bash
  git clone [https://link-to-project](https://github.com/Shakiqurrahman/computer-club-spi.git)
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
