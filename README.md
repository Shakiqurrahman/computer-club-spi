# Computer Club - SPI

## Tech Stack

**Client:** React, Tailwind CSS, Node JS, Express JS, MongoDB

## Run Locally

Clone the project

```bash
  git clone https://github.com/Shakiqurrahman/computer-club-spi.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
